Descripción
Este repositorio contiene el desarrollo de un mapa conceptual sobre el ambiente pedagógico y el territorio como escenario educativo, realizado como parte del curso Diseño de Ambientes de Aprendizaje (Código: 50003) 
El mapa conceptual representa la relación entre los elementos fundamentales que conforman un ambiente pedagógico significativo, incluyendo el territorio, el espacio educativo, la dotación, las interacciones, la cultura y la intencionalidad pedagógica.

Objetivo
Comprender la interrelación entre el territorio y el ambiente pedagógico como elementos clave en la construcción de experiencias educativas significativas, inclusivas y contextualizadas.
